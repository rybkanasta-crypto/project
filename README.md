## 2. Опис системи

### 2.1 Огляд
Описує загальну структуру та призначення системи.

### 2.2 Функціональні вимоги
- Система повинна дозволяти користувачам створювати облікові записи.
- Система повинна підтримувати автентифікацію через Google та Facebook.

### 2.3 Нефункціональні вимоги
- Продуктивність системи: час відповіді на запит не повинен перевищувати 2 секунд.
- Система повинна бути доступна 99.9% часу.
